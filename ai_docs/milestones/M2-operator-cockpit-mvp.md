# M2 - Operator Cockpit MVP

M2 uses the M1 read model to ship the first daily-usable Gastown-native cockpit for an operator working from convoys, beads, and contextual open flows.

## Included epics

- [EPIC-007 - Convoy cockpit MVP](../epics/EPIC-007-convoy-cockpit-mvp.md)
- [EPIC-008 - Bead inspector and ready-work surfaces](../epics/EPIC-008-bead-inspector-and-ready-work-surfaces.md)
- [EPIC-009 - Attention routing and contextual open flows](../epics/EPIC-009-attention-routing-and-contextual-open-flows.md)

## Exit criteria

- Convoys are the primary operator attention view.
- Ready work and bead detail are visible without leaving Gmux.
- Alerts and context jumps can open the correct workspace, worktree, and pane preset.

## Not in scope

- Mutating hooks or Beads from the UI.
- Deep persistence, review, or MCP work.
